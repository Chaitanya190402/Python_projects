# Voice-Controlled Assistant

This is a simple voice-controlled assistant script written in Python. The script uses the SpeechRecognition library to convert spoken words into text and performs various actions based on the recognized commands.

## Requirements
- Python 3.x
- SpeechRecognition library (`pip install SpeechRecognition`)
- webbrowser, os, and subprocess libraries

## Usage
1. Run the script.
2. Speak a command when prompted.
3. The assistant will perform actions based on the recognized command.

## Supported Commands
- Play a song on YouTube
- Perform a Google search
- Adjust system volume
- Adjust screen brightness
- Open Notepad or Spotify
- Play a specific video (e.g., Salaar-Part-1-Telugu)
